# RZ Automation Hub

Professional automation and AI workspace built with Next.js 14, featuring MacOS Sonoma-inspired UI design.

## Features

- **API Playground**: Test and debug REST APIs with full request customization
- **Workflow Automation**: Integrate with n8n for powerful workflow orchestration
- **AI Studio**: Powered by Google Gemini Flash 2.5 for AI capabilities
- **Execution Logs**: Monitor and troubleshoot workflow executions
- **Project Showcase**: Display your production applications

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- TailwindCSS
- shadcn/ui
- Zustand (optional)
- Axios
